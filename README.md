# libfido2

A WIP Rust wrapper for [libfido2](https://github.com/Yubico/libfido2).

## Current progress

- [x] Device detection
- [x] Device creation
- [x] Device CBOR information
- [x] Device setting management
- [x] Credential creation
  - [ ] Docs
- [ ] Credential verification
